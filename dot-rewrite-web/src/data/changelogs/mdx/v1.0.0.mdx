# .note Beta Launch

_March 28, 2025 • Version v1.0.0 • **Feature Update**_

---

We're excited to launch the **beta version of .note** — a modern way to study smarter, faster, and more collaboratively.

## What's New

- **Collaborative note-taking** – work on notes with classmates in real time
- **Real-time syncing across devices**
- **Markdown support** for rich formatting
- **Basic analytics** to track your study habits
- **Cross-platform compatibility** – web, tablet, and mobile

Welcome to your new AI-enhanced note companion. We're just getting started.
